# ai_nn_controller

A Python framework for building AI-driven network control applications with MCP (Model Context Protocol) integration.

## Installation

```bash
pip install ai_nn_controller
```

Or install from source:

```bash
pip install -e /path/to/ai_nn_controller
```

## Quick Start

### 1. Define Your Commands

Create a `commands.py` file to register your domain-specific commands:

```python
import json
from ai_nn_controller import register_command

def set_gain_handler(node_id: int, value: dict) -> str:
    """Handle SET_GAIN command."""
    return json.dumps({"target_gain": value.get("target_gain")})

# Register with custom schema
register_command(
    name="SET_GAIN",
    handler=set_gain_handler,
    schema={
        "description": "Set amplifier gain",
        "properties": {
            "node_id": {"type": "integer"},
            "target_gain": {"type": "number"}
        },
        "required": ["node_id", "target_gain"]
    }
)

# Or register with auto-generated default schema
register_command(
    name="RESET_NODE",
    handler=lambda node_id, value: json.dumps({"reset": True})
)
```

### 2. Create Your Application

```python
from ai_nn_controller import AicApp, AicController, aic_app

# Import commands to register them
import commands

@aic_app(name="MyNetworkApp")
class MyNetworkApp(AicApp):
    aic_app_id = 1
    control_loop_update_time = 2  # seconds

    # Measurements to read from nodes
    read_measurements = {
        1: ["power", "temperature"],
        2: ["power", "temperature"],
    }

    # Control functions available per node
    control_functions = {
        1: ["SET_GAIN"],
        2: ["SET_GAIN", "RESET_NODE"],
    }

    # Note: cell_ids and send_commands are auto-generated by @aic_app decorator
    # cell_ids will be: [1, 2] (union of read_measurements and control_functions keys)

    @classmethod
    def process(cls, measurements):
        """Process measurements and optionally send commands."""
        print(f"Received measurements: {measurements}")

        # Example: Send a command
        # cls.add_command(("SET_GAIN", {"node_id": 1, "value": {"target_gain": 20}}))

# Run with FastAPI server
if __name__ == "__main__":
    AicController(with_api=True).run()
```

### 3. Add Command Validators (Optional)

Command validators provide guardrails for commands sent via the REST API or MCP tools. They are completely optional - if no validator is defined, commands pass through unchanged.

```python
from ai_nn_controller import AicApp, aic_app, command_validator

@aic_app(name="MyNetworkApp")
class MyNetworkApp(AicApp):
    control_functions = {1: ["SET_GAIN"]}

    MAX_GAIN = 25.0  # Safety limit

    @command_validator("SET_GAIN")
    @classmethod
    def validate_set_gain(cls, params: dict) -> tuple[bool, str | None]:
        """Validate SET_GAIN commands before execution."""
        target_gain = params.get("target_gain", 0)
        if target_gain > cls.MAX_GAIN:
            return False, f"Gain {target_gain} exceeds max {cls.MAX_GAIN} dB"
        return True, None
```

**Note:** Validators are primarily useful when the FastAPI REST API or MCP tools are being used to send manual control commands.

### 4. Add Agent-Controlled Operations (Optional)

Agent-controlled operations let MCP/AI agents execute logic **inside the process loop** with access to live measurements. The return value goes back to the MCP caller.

```python
from ai_nn_controller import AicApp, aic_app, agent_controlled

@aic_app(name="SmartApp")
class SmartApp(AicApp):
    read_measurements = {8: ["preamp_gain"]}
    control_functions = {8: ["SET_GAIN"]}

    # IMPORTANT: @classmethod must be ABOVE @agent_controlled
    @classmethod
    @agent_controlled(
        name="optimize_gain",
        description="Optimize gain based on a strategy",
        schema={
            "properties": {
                "node_id": {"type": "integer"},
                "strategy": {"type": "string", "enum": ["max_snr", "min_power"]}
            },
            "required": ["node_id", "strategy"]
        }
    )
    def handle_optimize_gain(cls, request, measurements):
        """Runs inside the process loop with access to live measurements."""
        node_id = request["node_id"]
        latest = measurements.get(node_id, [{}])[-1] or {}
        new_gain = latest.get("preamp_gain", 15) + 2.0
        cls.add_command(("SET_GAIN", {"node_id": node_id, "value": {"target_gain": new_gain}}))
        return {"status": "applied", "new_gain": new_gain}

    @classmethod
    def process(cls, measurements):
        pass
```

This generates an MCP tool `SmartApp_optimize_gain`. The app must be `running` for the tool to work.

### 5. Declare Plugin Dependencies (Optional)

Apps can depend on reusable service plugins (e.g. storage, model registry, monitoring
adapters) built with the `plugin_framework` subpackage. Declare a dependency via
`required_plugins`; the controller validates it at startup and refuses to run if the
plugin isn't installed.

```python
from ai_nn_controller import AicApp, aic_app

@aic_app(name="MyNetworkApp")
class MyNetworkApp(AicApp):
    required_plugins = ["ConsolePlugin"]

    @classmethod
    def process(cls, measurements):
        console = cls.plugins.get("ConsolePlugin")
        if console:
            console.log_event("process_tick")
```

See `docs/user_guide/developing_plugins.rst` for the full guide on writing and
registering a plugin, and the repo-root `AGENT.md` for the complete
plugin-authoring workflow.

### 6. Run Your Application

```bash
python my_app.py
```

The FastAPI server will start at `http://localhost:8000` with:
- Swagger UI: `http://localhost:8000/docs`
- MCP endpoints for AI agent integration

## Features

- **Decorator-based app definition**: Use `@aic_app` to define your applications
- **Command registry**: Register custom commands with handlers and JSON schemas
- **MCP integration**: Auto-generated MCP tools for AI agents (Claude, etc.)
- **FastAPI REST API**: Built-in API for app management
- **ZeroMQ messaging**: High-performance pub/sub messaging with nodes

## API Reference

### Registry Functions

```python
from ai_nn_controller import (
    register_command,      # Register a single command
    register_commands,     # Register multiple commands
    has_command,           # Check if command exists
    execute_command,       # Execute a registered command
    list_commands,         # List all registered commands
    get_command_schema,    # Get JSON schema for a command
)
```

### Core Classes

```python
from ai_nn_controller import (
    AicApp,              # Base class for applications
    AicController,       # Controller that manages apps
    AicManager,          # Singleton manager for app registry
    aic_app,             # Decorator for app classes
    command_validator,   # Decorator for command validators (optional)
    agent_controlled,    # Decorator for agent-controlled operations (optional)
)
```

### AicController Options

```python
AicController(
    with_api=True,           # Start FastAPI server (default: False)
    api_host="0.0.0.0",      # API host (default: "0.0.0.0")
    api_port=8000,           # API port (default: 8000)
).run()
```
